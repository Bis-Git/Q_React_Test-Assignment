Ova aplikacija je moje rješenje za Q Agency React Coding skill Assignment.

Aproksimacija vremena: 3 - 4 h

Aplikacija preko Rest API-a pokupi podatke o postovima, korisnicima i komentarima, te ih prikazuje u obliku blog postova.

Homepage prikazuje listu postova i tražilicu koja pretražuje po tituli posta. Klikom na post otvara se novi tab u browseru i prikazuje detalje odabranog posta.

Svaka komponenta koristi custom hook nazvan useGreeter koji u konzoli prikazuje pozdravnu poruku svake komponente u formatu (Hello from ComponentName).
